# Test Case Automation Studio

AI-powered test case generator built with Claude API.

## Live Demo
https://SamiGotCurious.github.io/test-automation-studio/

## Features
- Upload CSV, Excel, JSON, or TXT datasets
- Custom requirements field for fine-grained control
- Functional, edge, negative, integration & performance tests
- Filter, search, and status tracking per test case
- Export results as CSV or JSON
- Live analytics dashboard

## Tech Stack
- Vanilla HTML/CSS/JS (single file, zero build step)
- Claude Sonnet 4 API for AI-powered generation
- SheetJS for Excel file parsing
- Hosted on GitHub Pages

## How to Use
1. Upload your dataset or paste requirements
2. Configure test types and priority mode
3. Add any custom requirements in the free-text box
4. Click Generate — Claude analyzes and builds your test cases
5. Filter, update status, and export results
